1. 웹사이트 URL을 가져오는 JavaScript 코드를 작성하라.

2. 파일의 확장자를 가져오는 코드를 작성하라. 예) images.jpg ==> jpg

3. 주어진 숫자가 13보다 적으면 13과의 차이값을 구하고 13보다 크면 절대차이의 두 배를 반환하는 코드를 작성하라.

4. 두 정수의 합을 계산하는 프로그램을 작성하라. 단 두 값이 같으면 합의 세배를 반환하라.

5. 두 개의 주어진 숫자를 확인하고 숫자중 하나가 50이거나 두 수의 합이 50이면 true를 반환하라.

6. 주어진 정수와 100 또는 400의 차의 절대값이 20 이내인지 확인하는 코드를 작성하라.

7. 주어진 두 수중 하나는 양수, 하나는 음수이면 true 를 출력하는 코드를 작성하라.
